プログラミング言語フレンズ Kemono
====

あなたはプログラミング言語フレンズなんだね！

Spec
----

Brainfuck系言語です

- *たのしー！* - ポインタを右へ移動
- *たーのしー！* - ポインタの指す値を1増やす
- *すごーい！* - ポインタを左へ移動
- *すっごーい！* - ポインタの指す値を1減らす
- *うわー！* ポインタの指す値が0なら 対応する 'わーい！' までジャンプする
- *わーい！* ポインタの指す値が0で無いなら 対応する 'うわー！' までジャンプする
- *なにこれなにこれ！* ポインタの指す値を出力する
- *おもしろーい！* 入力から1バイト読み込こむ


それ以外の文字や改行は全て無視されます
